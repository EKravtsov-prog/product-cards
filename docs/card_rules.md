# Rules for Product Cards

## Общие правила

1. Не придумывать характеристики без источника.
2. Ключевое слово начинать с существительного.
3. Для числовых значений использовать десятичную запятую.
4. Единицы измерения указывать в сокращённом виде: мм, м, кг, В, А, кА, Гц.
5. Размеры указывать через знак `×`.
6. Для габаритов использовать отдельные характеристики:
   - Длина;
   - Ширина;
   - Высота.
7. Не использовать обобщённую характеристику `Габариты`.
8. В полном наименовании после названия характеристики ставить двоеточие.
9. Основной разделитель характеристик — точка с запятой.

## Рекомендуемый порядок характеристик

1. Идентифицирующие признаки.
2. Тип / исполнение / серия.
3. Основные технические параметры.
4. Размеры.
5. Материал.
6. Покрытие.
7. Стандарт.
8. Производитель / бренд.

## Источники

В карточке желательно фиксировать:

- официальный сайт производителя;
- каталог;
- паспорт изделия;
- ГОСТ / DIN / ISO;
- проверенную карточку поставщика.

Если источник не найден — указать `Источник: не указан`.
