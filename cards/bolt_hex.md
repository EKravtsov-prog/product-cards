# Болт шестигранный

## 1. Краткое описание

Болт с наружной резьбой и шестигранной головкой, применяемый для разъёмных резьбовых соединений.

## 2. Ключевое слово

`Болт шестигранный`

## 3. Возможные синонимы

- Болт с шестигранной головкой
- Болт DIN 933
- Болт DIN 931
- Болт ГОСТ 7798
- Болт ГОСТ 7805

## 4. Классификационные признаки

| Признак | Значение / правило |
|---|---|
| Вид продукции | Болт |
| Назначение | Крепёж |
| Стандарт | ГОСТ / DIN / ISO |
| Материал | Сталь, нержавеющая сталь, латунь |
| Исполнение | Полная или неполная резьба |

## 5. Характеристики

| Характеристика | Обязательность | Единица | Пример значения | Комментарий |
|---|---:|---|---|---|
| Диаметр резьбы | mandatory | мм | M10 | Указывается как обозначение резьбы |
| Длина | mandatory | мм | 40 | Длина стержня |
| Шаг резьбы | optional | мм | 1,5 | Если указан в источнике |
| Класс прочности | optional |  | 8.8 | Для стальных болтов |
| Материал | optional |  | сталь | Без домыслов |
| Покрытие | optional |  | цинк | Если указано |
| Стандарт | optional |  | DIN 933 | Если указан |

## 6. Шаблон краткого наименования

```text
Болт шестигранный M[*]×L[*][!]
```

```text

```

## 8. Примеры нормализованных наименований

| Исходное наименование | Краткое наименование | Полное наименование |
|---|---|---|
| Болт DIN933 M10x40 кл.пр. 8.8 оцинк | Болт шестигранный M10×L40мм | Болт шестигранный; Диаметр резьбы: M10; Длина: 40 мм; Класс прочности: 8.8; Покрытие: цинк; Стандарт: DIN 933 |
| Болт ГОСТ 7798-70 М12х60 | Болт шестигранный M12×L60мм | Болт шестигранный; Диаметр резьбы: M12; Длина: 60 мм; Стандарт: ГОСТ 7798-70 |

## 9. Правила распознавания

| Условие | Результат |
|---|---|
| Есть слово `болт` и указан ГОСТ/DIN/ISO для шестигранных болтов | Классифицировать как болт шестигранный |
| Есть паттерн `M10x40`, `М12х60` рядом со словом `болт` | Извлечь диаметр резьбы и длину |
| Есть `кл.пр.`, `прочн.`, `8.8`, `10.9` | Извлечь класс прочности |

## 10. Исключения и спорные случаи

- Анкерные болты не относить сюда без проверки.
- Винты с внутренним шестигранником не относить сюда.
- Шпильки без головки не относить сюда.
