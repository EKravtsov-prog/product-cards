# Product Cards

Репозиторий для ведения карточек продуктов / материалов.

## Структура

```text
cards/       — карточки продуктов в Markdown
templates/   — шаблоны карточек
data/        — машинно-читаемый реестр карточек
schemas/     — JSON Schema для проверки структуры
scripts/     — служебные скрипты
docs/        — правила заполнения
```

## Как добавлять новую карточку

1. Скопировать файл `templates/product_card_template.md`.
2. Переименовать файл латиницей, например: `cards/bolt_hex.md`.
3. Заполнить карточку.
4. Добавить запись в `data/products.jsonl`.
5. Запустить проверку:

```bash
python scripts/validate_products.py
```

## Рекомендуемый процесс работы

- одна карточка = один продукт / вид продукции;
- изменения делать через отдельный commit;
- если карточка спорная — ставить статус `draft`;
- если карточка проверена — ставить статус `approved`.

## Статусы карточек

- `draft` — черновик;
- `review` — требует проверки;
- `approved` — утверждено;
- `deprecated` — устарело.
